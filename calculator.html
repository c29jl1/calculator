calculator abbreviations

  • ac: all clear
  • c: clear entry

Keyboard Shortcuts

  • enter: calculate result
  • backspace: clear entry or all clear
  • 0: zero
  • 1-9: numbers
  • . : decimal
  • + : addition
  • - : subtraction
  • * : multiplication
  • / : division
  • % : percentage
  • √ : square root
